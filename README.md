# 🤖 Rèz AI

Rèz is a personal AI assistant built with Python and Google's Gemini API.

The goal of this project is to gradually build a fully featured AI assistant capable of remembering conversations, understanding voice, seeing images, searching the web, and helping with everyday tasks.

---

## ✨ Current Features

* Chat with Google's Gemini 2.5 Flash
* Modular Python architecture
* Persistent chat history
* Git version control
* GitHub integration

---

## 🛠️ Built With

* Python 3.12
* Google Gemini API
* Git
* GitHub
* Visual Studio Code
